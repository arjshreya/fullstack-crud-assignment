Spring Boot Notes-

*Definition-Spring Boot is a framework for building applications in Java .

-Spring Boot makes it easy to create stand-alone, production-grade Spring based Applications that you can just run.

-Spring Boot reduces the effort required to set up and configure a Spring application.

-Spring Boot provides auto-configuration. It has embedded(in-built) tomcat server.

-Spring required installing webserver(tomcat) while deploying application manually.

-In Spring Boot, we use @SpringBootApplication annotation on main class. This single annotation replaces need for setting up a manual Spring application context.

-annotation-hint, instruction 
-bean-refers to an Object that is instantiated managed and maintained by spring IoC(no need to initialize every time)
-Application Context creates bean and gives you - this was manually done in spring but in spring boot it is automatically done.

-Spring initializer- a website created for convenience to create spring boot project.

-Maven - build tool automates and makes things easy
